    # 使用ASH 主题 请先满足ASH的要求



    #### 安装程序要求


    - [ ] 1. typecho程序

    - [ ] 2. typecho 1.2正式版（推荐）

    - [ ] 3. php需要7.3 版本 （更高版本没测试过）

    - [ ] 4. 请使用mysql数据库 （不支持sqlite）

    - [ ] 5. 确保主题文件夹名为ASH （否则会出事情的）

    - [ ] 6. 最后欢迎使用 ASH 主题 🎉


    ---


  • 版权问题

    • ASH主题全部功能来源joe主题，少部分来源网络或他人主题里，请使用者尊重下他人成果，禁止搬用。ASH 主题装修来源各大博主博客的样式结合而来。


    ASH 主题支持更多人参与二开与拓展，但禁止出现ASH 二改之后变卖事件。




    ASH 来源



    ASH 主题由 joe 主题修改而来，由于 joe 主题支持二开并且功能多，受多人喜爱，在网上能看到不少人二开 joe 主题，修改主题名字并且不低于三十出售，而且也不少人愿意购买，这不就是赚钱的机会吗？于是我开始向往他们，我看了很多类似 joe 主题或 joe 修改过的主题，结合了一个主题且命名为 ASH（灰），二十岁是人生灰暗的开始，迷茫没有方向，不自量力的认为没有自己做不好的事，一度认为不放弃就有希望，到头来的一切都是竹篮打水。


    当这个 ASH 主题开始二改的时候，会想当初为什么选择了 typecho ，好像是因为容易搭建，好像是因为主题更简约，好像更是因为不需要很贵的服务器。玩博客的目的是为了释放自己的那些不开心，讲述那些不知道和谁分享的事，记录过去舍不得遗忘且因时间的流逝只能忘却的点点滴滴。


    后来逐渐去追求更好看或者更简约的主题和少见或特殊的域名，买的前三天喜欢的不得了，做梦都会笑，到后面，发现了更好的，也可能是拥有了，喜欢淡了，后悔购买，然后又开始买了一个喜欢的主题或域名…持续循环，相信不少人和我一样。当然肯定也会有不少人会说着：“付费的啊，那算了，没那个钱。”然后就会被怼说：“没钱你玩什么博客。”其实没人规定说没有钱就不能玩博客，不能玩 typecho 的。也有少部分专一的，看中之后就会用个一年半载，但更多的是觉得没拥有之前喜欢不得了，拥有后就觉得也不过如此。


    所以 ASH 主题不采取任何形式出售，将免费发布供每个人使用，虽说这样相当于自己又放弃了一个可以赚钱的机会，但比起付费，我更喜欢友情赞助，毕竟赞助是心意，出售是索取，（避个嫌）个人觉得非自己原创出售的物品都是索取。


    ASH 以这样的方式出现在大家的面前，只希望喜欢的人能更喜欢，不喜欢的人能不讨厌。


    最后还是要再重复一边，使用ASH主题者，请不要让ASH参与到买卖事件中。支持二改，但不支持出售。介意者可以不用，不改。
